# 高考倒计时

> 🎓 一款精美的 WPF 桌面倒计时应用，为高考冲刺加油！

## ✨ 功能

- ⏳ **实时倒计时** — 天/时/分/秒精确刷新
- 🌐 **中英双语** — 中文 + 英文双行倒计时显示
- 📊 **高中进度条** — 可视化展示高中生活已过百分比
- 🎨 **深度自定义** — 字体、字号、颜色、透明度均可调
- 📐 **屏幕定位** — 6 种预设位置 + 偏移量 + 始终置顶
- 🎬 **数字脉冲动画** — 每秒轻量缩放脉冲（可开关）
- 💾 **设置持久化** — JSON 配置文件，重启不丢失

## 🚀 快速开始

### 运行

```bash
dotnet run
```

### 编译发布（64 位独立可执行文件）

```bash
dotnet publish -c Release -r win-x64 --self-contained true
```

发布输出位于 `bin/Release/net8.0-windows/win-x64/publish/`。

## 📁 项目结构

```
高考倒计时/
├── MainWindow.xaml       # 主窗口 UI
├── MainWindow.xaml.cs    # 主窗口逻辑（倒计时、动画）
├── SettingWindow.xaml    # 设置窗口 UI
├── SettingWindow.xaml.cs # 设置窗口逻辑
├── Settings.cs           # 设置数据模型（JSON 持久化）
├── App.xaml / .cs        # 应用入口
└── 高考倒计时.csproj      # 项目文件
```

## 📝 设置说明

| Tab | 功能 |
|-----|------|
| 🎨 外观 | 字体、字号、数字/文字颜色、进度条颜色、透明度 |
| 📐 位置 | 屏幕位置预设、偏移量、始终置顶 |
| 👁 显示 | 英文行、进度条、百分比文字开关、小数位数 |
| ✏ 文本 | 中英文提示文本自定义 |
| 📅 日期 | 高考目标日期、进度起算日期 |
| ✨ 动画 | 脉冲动画、设置窗口动画开关 |
